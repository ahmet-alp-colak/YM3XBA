**Sistem:** Bilişsel RF Spektrum Haritalama Sistemi

## 📍 Genel Bakış

Bu kılavuz, **1 km² (1000m × 1000m)** kısa menzil taktik saha operasyonları için ED istasyon yerleşim stratejilerini, optimal baseline (baz çizgisi) mesafelerini ve sistem konfigürasyon parametrelerini içermektedir.

### Senaryo Özeti
- **Saha boyutu:** 1000m × 1000m (~1 km²)
- **Hedef mesafesi:** 0 - 1500m (saha merkezi referanslı)
- **İstasyon sayısı:** 2 adet ED istasyonu (minimum)
- **Beklenen hassasiyet:** ±50m - ±150m (SNR ve geometriye bağlı)
- **Kullanım alanları:** Taktik tatbikatlar, kısa menzil SIGINT, sınır güvenliği

## 🎯 İstasyon Yerleşim Stratejisi

### 1. **Optimal Yerleşim: L-Konfigürasyonu (Tavsiye Edilen)**

```
┌─────────────────────────────────────────────┐
│                                             │
│         OPERASYON SAHASI (1 km²)            │
│                                             │
│             500m                            │
│        ↕                                    │
│   ┌────────────────┐                        │
│   │                │                        │
│   │      [T]       │   T: Hedef             │
│   │   (0, 0)       │   S1: İstasyon 1       │
│   │                │   S2: İstasyon 2       │
│   └────────────────┘                        │
│        ↕                                    │
│       500m                                  │
│                                             │
│  S1 ────────────────────────────── S2       │
│  (-707m, -707m)          (707m, -707m)      │
│                                             │
│  Baseline: 1414m (√2 km)                    │
│  Kesişim Açısı: ~90° (Optimal)              │
│  GDOP: ~1.5 (Mükemmel)                      │
│                                             │
└─────────────────────────────────────────────┘
```

**Koordinat Örnekleri (WGS84):**
- **Saha Merkezi (Origin)**: 39.0000°N, 32.0000°E
- **İstasyon 1 (S1)**: 39.0000°N - 0.0064°, 32.0000°E - 0.0089° → ~(38.9936°N, 31.9911°E)
- **İstasyon 2 (S2)**: 39.0000°N - 0.0064°, 32.0000°E + 0.0089° → ~(38.9936°N, 32.0089°E)

**Avantajları:**
- ✅ Kesişim açısı ~90° (ideal GDOP)
- ✅ Saha merkezine eşit uzaklık
- ✅ 360° kapsama
- ✅ Multipath etkisi minimum
- ✅ Setup süresi kısa

**Dezavantajları:**
- ⚠️ Sahaya göre geniş baseline (lojistik)
- ⚠️ Güney sektörde hassasiyet azalması

## 📏 Baseline (Baz Çizgisi) Mesafeleri

### Optimal Baseline Seçimi

| Hedef Mesafesi | Minimum Baseline | Optimal Baseline | Maksimum Baseline | Beklenen RMS Hata |
|----------------|------------------|------------------|-------------------|-------------------|
| 0 - 500m       | 400m             | 800m             | 1500m             | ±30m - ±80m       |
| 500 - 1000m    | 600m             | 1200m            | 2000m             | ±50m - ±120m      |
| 1000 - 1500m   | 800m             | 1500m            | 2500m             | ±80m - ±180m      |

### Baseline Hesaplama Formülü

```
Optimal_Baseline = Target_Range × sin(60°) × 2
                 ≈ Target_Range × 1.732
```

**Örnek:**
- Hedef merkez nokta (500m): `Baseline = 500 × 1.732 ≈ 866m`
- Hedef maksimum (1500m): `Baseline = 1500 × 1.732 ≈ 2598m`

### Baseline - GDOP İlişkisi

| Baseline / Hedef Mesafesi | Kesişim Açısı | GDOP | Durum |
|---------------------------|---------------|------|-------|
| > 1.5                     | 60-120°       | 1.0-2.0 | ✅ Mükemmel |
| 1.0 - 1.5                 | 40-60°        | 2.0-3.5 | ✅ İyi |
| 0.5 - 1.0                 | 30-40°        | 3.5-6.0 | ⚠️ Kabul Edilebilir |
| < 0.5                     | < 30°         | > 6.0 | ❌ Kötü |

## 📊 Performans Beklentileri

### RMS Hata Tahmini (1 km² Saha)

| Senaryo | Baseline | Kesişim Açısı | AoA Hata | RMS Hata (Tahmini) | GDOP |
|---------|----------|---------------|----------|---------------------|------|
| **İdeal** | 1400m | 90° | ±5° | ±30m - ±60m | 1.5 |
| **İyi** | 1200m | 70° | ±10° | ±50m - ±100m | 2.3 |
| **Kabul Edilebilir** | 800m | 50° | ±10° | ±80m - ±150m | 3.8 |
| **Kötü** | 600m | 35° | ±15° | ±150m - ±300m | 6.2 |
| **Kullanılamaz** | 400m | 20° | ±15° | ±500m+ | >10 |

### RMS Hata Formülü (Yaklaşık)

```
RMS_Error ≈ (Range × tan(AoA_Error)) / GDOP_Factor

Örnek (İyi Senaryo):
- Range = 1000m
- AoA_Error = 10° (±10°)
- GDOP = 2.3

RMS = (1000 × tan(10°)) / 2.3
    ≈ (1000 × 0.176) / 2.3
    ≈ 176 / 2.3
    ≈ 76m
```

## 🚀 Kurulum Prosedürü

### Adım 1: Saha Keşfi
1. ✅ Operasyon sahasını belirle (1 km²)
2. ✅ Saha merkezini GPS ile işaretle
3. ✅ Arazi tipini analiz et (düz, tepelik, ormanlık)
4. ✅ RF yansıma kaynaklarını tespit et (binalar, metal yapılar)

### Adım 2: İstasyon Konumlandırma
1. ✅ **L-Konfigürasyonu** için:
   - Saha merkezinden 45° ve 135° yönlere 1000m git
   - GPS koordinatlarını kaydet
2. ✅ **Kompakt Yerleşim** için:
   - Saha güney kenarından ±400m offset
   - Baseline 800m kontrol et

### Adım 3: Sistem Kalibrasyonu
1. ✅ Her istasyonda GPS GPSDO zaman senkronizasyonu kontrol
2. ✅ Antenna array kalibrasyonu (4.4 DF modülü)
3. ✅ RF ortam gürültü seviyesi ölç (noise floor)
4. ✅ Referans kaynak (known emitter) ile doğrulama

### Adım 4: Operasyonel Test
1. ✅ Bilinen konumda test yayıcı yerleştir
2. ✅ Geolocation sonuçlarını karşılaştır
3. ✅ RMS hata <200m olduğunu doğrula
4. ✅ GDOP <5.0 olduğunu kontrol et
